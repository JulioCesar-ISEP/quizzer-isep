# Glossário

| Termo | Definição |
|-------|-----------|
| **SubjectType** | Classificação da disciplina: `CALCULUS`, `DISCRETE_MATH`, `CODE`, `SYSTEMS` |
| **Survival Mode** | Minijogo 4 — simulação de exame com cronómetro e bloqueio ao errar (roadmap) |
| **Payload** | `SurvivalSessionPayload` — perguntas + metadados gerados pelo RAG ou adaptador |
| **Shell** | Componente raiz da UI que compõe vistas e hooks |
| **Adaptador estático** | Ponte temporária `data/prcmp` → contrato Survival |
