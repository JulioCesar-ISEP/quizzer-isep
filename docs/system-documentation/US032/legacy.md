# US032 — Dashboard global com gráficos e heatmaps

**Épico:** Advanced / Full  
**Phase:** Phase 3 — Advanced  
**Prioridade:** Baixa  
**Status:** Planeado

## Resumo

Dashboard global agregando todas as cadeiras: progresso, consistência, weak points e tendências.

- Visão global multi-cadeira.
- Gráficos/heatmaps de sessões e acerto.
- Filtros por período e tipo de atividade.

## Ports

- Entrada: `GetGlobalDashboardUseCase`
- Saída: `IStatsRepository`
