# US025 — Tutoriais interativos dos minijogos

**Épico:** Enhanced  
**Phase:** Phase 2 — Enhanced  
**Prioridade:** Baixa  
**Status:** Planeado

## Resumo

Onboarding interativo por minijogo (Diff/Molde/Follow Bit/Survival) para reduzir atrito.

## Requisitos

- Cada minijogo tem um tutorial curto (1–2 min).
- Tutorial pode ser repetido e é guardado “já visto”.
- Acessível a partir do menu/ajuda.

## Ports

- Entrada: `StartTutorialUseCase`
- Saída: `ITutorialStateRepository`
