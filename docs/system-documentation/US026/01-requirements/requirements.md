# US026 — Requisitos

### User story

Como estudante, quero alternar entre tema claro/escuro e ter uma UI consistente para estudar com conforto e acessibilidade.

### Critérios de aceitação

- Alternar dark/light e persistir preferência.
- Acessibilidade mínima: contraste, foco, navegação por teclado.
- Ajustar copy e componentes para consistência.
